1. \\wsl$\Ubuntu\home\xinbang05\ragflow（Windows 文件管理器里输入这个路径）可以打开WSL Ubuntu文件。

2. WSL -d Ubuntu 进入Ubuntu系统

3. 启动ragflow服务

```
cd ~/ragflow
source .venv/bin/activate
export PYTHONPATH=$(pwd)
bash docker/launch_backend_service.sh
```

4. 用vscode打开WSL后端服务

```
-- 在 WSL 终端里：
cd ~/ragflow
code .
cursor . 
```
