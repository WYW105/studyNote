# NoSQL数据库和SQL数据库的关系

1. SQL是结构化、关系型数据库、使用SQL查询、支持ACID事务特性、磁盘存储、垂直扩展；
2. NoSQL是非结构化、非关系型数据库、非SQL查询、内存存储、水平扩展。
